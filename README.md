## 📦 설치 방법 (3단계)

### 1단계 — 압축 해제
다운로드한 ZIP 파일을 `C:\claudeprojects\hrmanage\` 폴더에 압축 해제하세요.

```
C:\claudeprojects\hrmanage\
  ├── 실행.bat          ← ★ 이 파일을 더블클릭하면 끝!
  ├── package.json
  ├── server\
  │   └── index.js
  ├── public\
  │   ├── index.html
  │   ├── css\style.css
  │   └── js\...
  └── data\             ← DB 파일 자동 생성됨
```

### 2단계 — Node.js 설치 확인
명령 프롬프트(cmd)에서:
```
node -v
```
버전이 표시되면 OK. 없다면 https://nodejs.org 에서 LTS 버전 설치.

### 3단계 — 실행
`실행.bat` 파일을 더블클릭하면:
1. 패키지 자동 설치 (최초 1회)
2. 서버 시작
3. 브라우저 자동 열림 → http://localhost:3000

## 🔄 테스트 시나리오 (추천 순서)

### 시나리오 A — 전체 평가 사이클
1. **한개발** (dev3) 로그인 → 목표 설정 → 승인 요청

4. 권한 관리에서 일반사용자를 관리자로 변경
5. 감사 로그 확인

## 🗄️ 데이터베이스
- 위치: `data\hrmanage.db` (SQLite 파일)
- 모든 민감 데이터는 AES-256-CBC 암호화 저장
- 삭제하면 초기화됩니다

## 🔧 포트 변경
기본 포트가 3000인데 충돌 시:
```
set PORT=4000 && node server\index.js
```
또는 `실행.bat`에서 `node server\index.js` 앞에 `set PORT=4000 &&` 추가

## ❓ 자주 묻는 질문

**Q: "node_modules 설치 오류"가 뜹니다**
A: 관리자 권한으로 cmd 실행 후 `npm install` 명령어 입력

**Q: 브라우저에서 접속이 안 됩니다**
A: 방화벽에서 3000번 포트 허용 또는 포트 변경

**Q: 데이터를 초기화하고 싶습니다**
A: `data\hrmanage.db` 파일 삭제 후 재실행
